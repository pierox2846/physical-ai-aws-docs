# 왜 지금 Physical AI인가?

Physical AI는 새로운 주제가 아닙니다. 최근에 **성공 확률이 급상승**했습니다.

### 촉발 요인

* **하드웨어 비용 감소 및 센서 기술 발전**
* **Foundation model**: 인지와 표현 학습 비용이 크게 감소했습니다.
* **멀티모달**: 텍스트-이미지-비디오-행동 시퀀스를 같이 다룹니다.
* **데이터 플라이휠**: 운영 데이터가 학습 성능을 끌어올립니다.
* **시뮬레이션 고도화**: 합성 데이터와 대규모 시나리오 테스트가 가능해졌습니다.
* **GPU/가속기 확산**: 학습과 추론의 단가가 내려갔습니다.
* **엣지/클라우드 결합**: 실시간은 현장, 학습은 클라우드가 표준입니다.

### 비즈니스가 관심 갖는 이유

* 인력 부족을 보완합니다.
* 품질 편차를 줄입니다.
* 위험 작업을 대체합니다.
* 운영 데이터를 자산화합니다.

### 현실적인 한계(지금도 어려운 것)

* 데이터 품질과 커버리지가 부족합니다.
* 안전 규격과 책임 소재가 복잡합니다.
* 도메인 갭이 여전히 큽니다.

{% hint style="info" %}
Physical AI는 ‘모델’만의 문제가 아닙니다. 데이터, 검증, 배포, 관측까지 포함한 **시스템 문제**입니다.
{% endhint %}
