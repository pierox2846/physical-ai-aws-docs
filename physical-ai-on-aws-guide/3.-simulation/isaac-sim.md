# Isaac Sim 환경 구축

Isaac Sim은 로봇 시뮬레이션과 합성 데이터 생성에 많이 씁니다. AWS에서는 보통 **GPU 인스턴스** 위에 띄우고 원격으로 접속합니다.

### AWS에서의 대표 실행 형태

* 단일 노드 개발: EC2 GPU 1대.
* 배치 시뮬/합성 데이터: 여러 노드로 수평 확장.
* 쿠버네티스 운영: EKS로 잡 단위 실행.

### 운영 설계 포인트

* 렌더링 워크로드는 GPU가 필수인 경우가 많습니다.
* 결과물(비디오/라벨/로그)은 S3에 모으는 게 편합니다.
* 시나리오 정의는 코드로 버전 관리하세요.
