# NVIDIA GR00T N1

[https://github.com/NVIDIA/Isaac-GR00T](https://github.com/NVIDIA/Isaac-GR00T)

GR00T는 NVIDIA의 로보틱스 파운데이션 모델 흐름을 보여주는 프로젝트입니다. 여기서는 “AWS에서 어떻게 다루면 좋은가” 관점만 정리합니다.

### 무엇을 할 때 참고하나

* 로봇 정책 학습/파인튜닝의 레퍼런스를 보고 싶을 때.
* Isaac Sim 기반 데이터/시뮬 워크플로우를 엮고 싶을 때.

### AWS에서의 배치 포인트

* 학습: SageMaker 또는 EKS(GPU 노드).
* 데이터: S3(원천/가공/데이터셋 분리).
* 실험/평가: 파이프라인으로 자동화.

### 체크리스트

* 어떤 입력(비전/상태/언어)을 쓰는가.
* 어떤 출력(연속 제어/행동 토큰)을 쓰는가.
* 평가를 시뮬/현실에서 어떻게 나눌 것인가.

{% hint style="info" %}
프로젝트/모델은 빠르게 변합니다. ‘특정 버전 고정’과 ‘재현 가능한 파이프라인’이 더 중요합니다.
{% endhint %}
