# 2. 학습 (Training)

Training은 모델 학습만이 아닙니다. **실험 추적, 평가, 재현, 승인**이 포함됩니다.

### 학습 대상

* 인지 모델: detection/segmentation/pose.
* 세계 모델: 비디오 예측, 상태 전이
* 정책 모델(RFM 포함): 관측 → 행동.

### 핵심 요구

* 분산 학습과 데이터 처리량
* 실험 추적과 파라미터 관리
* 모델 레지스트리와 릴리즈 승인
* 오프라인 평가 + 시뮬 평가
